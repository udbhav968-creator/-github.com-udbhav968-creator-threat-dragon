# StepCode - Academic Activity Tracking & Student Employability Platform

Welcome to the **StepCode** repository! StepCode is a dual-model SaaS platform designed to bridge the gap between institutional curriculum and real-world employability metrics.

## 🚀 Features

- **Dual-Mode Experience**: Seamlessly toggle between a Student (B2C) and University (B2B) interface.
- **Interactive Employability Mind-Map**: A dynamic, animated SVG-based tech tree that showcases learning pathways (YouTube Learning, React Course, GitHub Projects, Real-World Projects, Build Skills, DSA Practice).
- **Simulated SaaS Dashboard**: Experience a real-time mock institutional portal that tracks batch-wise milestone completions, mentor interactions, and active student streaks.
- **Multi-Step Onboarding**: A beautiful, interactive signup wizard that dynamically customizes the workspace based on user roles (Student vs. Educator) and career tracks.
- **Premium Aesthetics**: Built with a deep carbon dark mode, neon crimson accents (`#e52b2b`), and rich micro-interactions like 3D hover-tilt cards and flowing data paths.

## 🛠️ Technology Stack

This landing page and interactive demo is built completely dependency-free using vanilla web technologies to ensure maximum performance and minimal overhead:

- **HTML5**: Semantic layout and inline SVGs for crisp, scalable graphics.
- **CSS3**: Custom variables, Grid/Flexbox layouts, and advanced `@keyframes` animations (floating nodes, glowing borders, pulsing paths).
- **JavaScript (Vanilla)**: DOM manipulation, event handling, intersection observers, and simulated real-time data feeds.

## 🚦 Running Locally

To view the interactive landing page locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/udbhav968-creator/-github.com-udbhav968-creator-threat-dragon.git
   ```
2. Navigate into the project directory:
   ```bash
   cd -github.com-udbhav968-creator-threat-dragon
   ```
3. Start a local server (e.g., using Python):
   ```bash
   python -m http.server 8080
   ```
4. Open your browser and visit: `http://localhost:8080/`
